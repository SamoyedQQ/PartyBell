# PartyBell 🔔

FF14 台服(繁中服)Dalamud 插件:掛機發招募時,隊伍有任何變動都透過 **Discord Webhook** 即時通知你(可 @ 提及);招募到期(60 分鐘)前**自動重新發布**;排到副本還能**自動接受**。

## 功能

| 功能 | 說明 |
|---|---|
| ✅❌ 隊員加入 / 退出 | 跨界與本地隊伍皆支援,顯示名字@伺服器與職業 |
| 🎉 隊伍滿員 (8人) | 補滿 8 人時通知(預設會 @ 提及你) |
| 💬 隊伍頻道訊息 | Party / 跨界組隊 / 團隊頻道 |
| 📨 密語 (Tell) | 其他玩家密你時轉發內容 |
| ⚔️ 副本排到 (Duty Pop) | 任務搜尋器配對成功或招募人齊確認(預設會 @ 提及你) |
| ✅ 副本自動接受 | 排到副本後延遲 N 秒自動按「出發」 |
| 🔄 招募自動刷新 | 到期前 N 分鐘自動重新發布招募,成功/失敗回報到 Discord |

所有通知類型皆可個別開關與設定是否 Tag(@ 提及),並可設定「只在 AFK 時通知」「偵測到變動時在遊戲內跳出視窗」。

## 安裝

1. 遊戲內輸入 `/xlsettings` → **實驗性功能** 分頁
2. 在 **自訂插件倉庫** 加入以下網址後按 **+** 與 **儲存**:
   ```
   https://raw.githubusercontent.com/SamoyedQQ/TCRankingViewer/main/repo.json
   ```
3. `/xlplugins` → 搜尋 **PartyBell** → 安裝
4. 輸入 `/pbell` 開啟設定視窗

## 快速上手

### 1. 建立 Discord Webhook

1. 在你的 Discord 伺服器選一個頻道 → **編輯頻道** → **整合** → **Webhook** → **新 Webhook**
2. 複製 Webhook URL

### 2. 設定插件

1. 遊戲內輸入 `/pbell` 開啟設定 → **Webhook** 分頁
2. 貼上 Webhook URL → 按 **發送測試訊息**,Discord 頻道收到訊息即設定完成

### 3. 設定 @ 提及(選用)

想讓通知真的「響」手機的話:

1. Discord → 使用者設定 → 進階 → 開啟 **開發者模式**
2. 右鍵點自己的頭像 → **複製使用者 ID**
3. 貼到插件 Webhook 分頁的 **Tag 目標** 欄位(也可填 `everyone` 或 `here`)
4. 在 **通知** 分頁勾選想要 @ 的通知類型旁的 **Tag**

### 4. 自動化(選用)

**自動化** 分頁裡:

- **招募自動刷新**:發布招募後放著即可,插件會在到期前(預設 5 分鐘)自動重新發布,並回報到 Discord
- **副本自動接受**:排到副本後延遲 N 秒(預設 5 秒)自動按「出發」;延遲期間手動按掉則不動作

## 招募自動刷新注意事項

- 依賴遊戲內部函式(signature scan)與 UI 操作,**遊戲版本更新後可能失效**;失效時設定視窗會顯示警告並自動停用
- 刷新時會短暫開啟招募視窗(詳情 → 條件 → 重新招募),掛機時屬正常現象
- 戰鬥中 / 過圖 / 過場時會延後 1 分鐘再試;連續失敗 3 次會停止並發 Discord 警告(可設定 Tag)
